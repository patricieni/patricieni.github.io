Welcome to OT Metric Learning documentation!¶

mml is written by Patric

Quick Start:
  1. Install requirements.txt via conda
  2. Edit your Multiscale Metric Learning-GPU.ipynb
  3. Edit your index.rst and add the names of your *.ipynb files to the toctree.
  4. Browser access to doc html using “/otbuild/html/index_ot.html”
