Quick Start:
    #. Install ``requirements.txt`` via ``conda``

    #. Edit your ``Multiscale Metric Learning-GPU.ipynb``
    #. Edit your ``index.rst`` and add the names of your ``*.ipynb`` files
       to the ``toctree``.
    #. Browser access to doc html using "/otbuild/html/index_ot.html"
