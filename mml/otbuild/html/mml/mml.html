mml.gml.similarity_matrix(X, Y, k)[source]¶

Builds a similarity matrix using X and Y labels as well as k neighbours. Similarity is given by same labels and the similarity value is proportional to size of data n and neighbours k. The opposite is true for dissimilarity, same value with a minus sign.

Parameters:
  • X (array(float)) – Dataset
  • Y (array(float)) – Labels
  • k (int) – Number of neighbours used in GML
Returns:

A matrix w of shape X.shape[0],X.shape[0]
Return type:

array[float,float]
